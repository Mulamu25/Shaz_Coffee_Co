# SHAZ COFFEE CO.

## Project Description

SHAZ COFFEE CO. is a conceptual coffee shop website developed as part of the WEDE5020 Web Development module. The website showcases the brand's identity, menu offerings, and provides customers with essential information such as contact details, location, and an enquiry form.

The brand is inspired by modern café culture, focusing on comfort, community, and premium beverages.

---

## How to View the Website

1. Clone or download this repository to your local machine
2. Open the `index.html` file in any web browser (Chrome, Edge, Firefox)
3. Navigate through the pages using the menu links

**Live Demo:** [SHAZ COFFEE CO. on Netlify](https://shaz-coffee-co.netlify.app)

## Features

### Part 1: HTML Structure
- 5 fully linked HTML pages
- Consistent navigation menu on all pages
- Enquiry form with required field validation
- Embedded Google Maps iframe on contact page
- Customer favourites section on homepage
- Menu page with organised categories (Coffee, Cold Drinks, Snacks)
- Back to Top links on all inner pages

### Part 2: CSS Styling & Responsive Design
- External CSS stylesheet linked to all pages
- CSS Reset for cross-browser consistency
- CSS Variables for color palette management
- Poppins font from Google Fonts
- Flexbox navigation (horizontal desktop, vertical mobile)
- CSS Grid footer (3 columns desktop → 2 tablet → 1 mobile)
- Active page highlighting (orange text + underline)
- Logo rotation animation (360° on hover)
- Sticky header with position: sticky
- Button pseudo-classes (:hover, :active, :focus)
- Media queries for tablet (1024px) and mobile (768px)
- Responsive images with max-width: 100%
- Burnt orange glow effect on menu images
- Rounded corners on footer matching header

### Part 3: JavaScript Functionality, SEO & Deployment
- Animated hamburger menu icon (☰ → ✕) for mobile
- Sidebar menu that slides in from left (280px width)
- Menu overlay with dimmed background
- Welcome modal popup on homepage
- Image lightbox gallery for menu items
- Accordion FAQ section on About page
- Search and filter functionality for menu items
- Back to Top button with smooth scroll
- JavaScript form validation for enquiry page
- Real-time error messages
- Email format validation
- Form processing with email summary display
- Live date and time display in footer (updates every second)
- SEO title tags on all 5 pages
- SEO meta descriptions on all pages
- SEO meta keywords on all pages
- Descriptive alt text on all images
- robots.txt file for search engine crawlers
- sitemap.xml for search engine indexing
- Website deployed on Netlify
